> 커다란 세계를 담을 작은 서랍장

캐비닛키는 피치타르트와 함께하는 창작 세계관 공유 서비스입니다. 

다음과 같은 서비스의 영향을 받아 제작되었습니다.

* Notion
* Misskey
* Storynation
* ...

캐비닛키의 소스 코드는 공개됩니다.

* [리포지토리](https://github.com/jyhyun1008/cabinetkey-nuxt)

캐비닛키는 다음과 같은 프레임워크 및 환경에서 개발됩니다.

* Nuxt 3
* Node.JS
* PostgresQL
